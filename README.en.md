## 🌱 项目开发启动

```bash
# 克隆项目
git clone https://github.com/Wayne1308/wayUI.git

# 切换分支
git checkout dev # 开发迭代版本主分支

# 安装依赖
pnpm install
# 或
yarn install

# 本地开发 启动项目
pnpm run dev
# 或
yarn run dev
```
